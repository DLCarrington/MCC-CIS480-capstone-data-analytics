# MCC-CIS480-capstone-data-analytics
Final Capstone Project for Data Analytics Course

Overview
This research and analysis project investigated available data sources to provide results indicating the most productive areas of the U.S. to focus an advertising campaign promoting the purchase of vehicles that use renewable energy and the purchase of renewable energy systems.  Additionally, it investigated possible correlating data points to target households with enough income to purchase new electric vehicles (EVs) and solar systems.  The project included the application of data analysis methods and technologies, as well as the creation of visualizations for interpreting results.

This project empolyed Excel, and Power BI

Files are contained in folders named data, Power BI, and documentation
